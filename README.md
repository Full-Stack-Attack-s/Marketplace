## 1. Как брать задачи с доски?
[Переходите на доску](https://github.com/orgs/Full-Stack-Attack-s/projects/5/views/2), там будет 3 раздела: 
- Задачи связанные со мной (Т.е. задачи в которых испольнителем указаны вы)
- Распределение задач (Распределние задач по статусам)
- Обозреватель задач (Список всех задач, создан для назначения приоритета задачам)
  Вы ознакамливаетесь с задачей перейдя по её названию в карточку задания, как только ознакомились, переходите в обозреватель задач, и там в Assignees выбираете себя.
## 2. Как приступить к выполнению задачи правильно?
Чтобы сделать все верно вы должны выполнить несколько шагов.
- Перенести задачу в In progress (делаем по 1 задачи параллельно!)
- Создать ветку с номером задачи например: "24-paginations".
- При использовании общий файлов таких как: index.html, views.py и так далее предупреждать, что вы взяли файл в работу.
- При выполнении работы делать commit также с номером задачи например: "Сделал кнопки пагинации #24".
- Идешь на GitHub. Там появится желтая плашка с твоей веткой.
- Жмешь зеленую кнопку Compare & pull request.
- В описании пишешь магическую фразу Resolves #(номер задачи).
